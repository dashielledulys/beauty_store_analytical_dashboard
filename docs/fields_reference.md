## Структура справочника полей

### Исходные поля из Excel-таблиц:

**Датасет Products**:

**SKU** – уникальный id товара
**Категория** – категория товара
**Предмет** – наименование предмета (подкатегория/тип)
**Бренд** – бренд-производитель
**Цена (Декабрь)** – средняя цена за продукт в декабре 2023
**Цена (Ноябрь)** – средняя цена за продукт в ноябре 2023
**Продажи (Декабрь)** – объем продаж в декабре 2023 (в единицах товара)
**Продажи (Ноябрь)** – объем продаж в ноябре 2023 (в единицах товара)
**Выручка (Декабрь)** – объем выручки (₽) в декабре 2023
**Выручка (Ноябрь)** – объем выручки (₽) в ноябре 2023
**Средний % выкупа (Декабрь)** – доля выкупленных товаров от числа заказанных в декабре 2023
**Средний % выкупа (Ноябрь)** – доля выкупленных товаров от числа заказанных в ноябре 2023
**Страна** – страна происхождения товара

**Датасет Dynamic**:
**Бренд** – бренд-производитель
**Месяц** – дата начала месяца отчетности
**Продажи** – объем продаж за месяц (в единицах товара)
**Выручка** – объем выручки за месяц
**Средний % выкупа** – доля выкупленных товаров от числа заказанных за месяц

### Вычисляемые поля датасета (бизнес-логика)

**Датасет Products**:

**Прирост выручки** – процент изменения выручки декабрь к ноябрю

```
IF IFNULL(SUM([Выручка (Ноябрь)]), 0) != 0 
THEN (SUM([Выручка (Декабрь)]) - SUM([Выручка (Ноябрь)])) / SUM([Выручка (Ноябрь)]) 
END
```

**Прирост продаж** – процент изменения продаж декабрь к ноябрю

```
IF IFNULL(SUM([Продажи (Ноябрь)]), 0) != 0 
THEN (SUM([Продажи (Декабрь)]) - SUM([Продажи (Ноябрь)])) / SUM([Продажи (Ноябрь)]) 
END
```

**Изменение цены** – процент изменения средней цены декабрь к ноябрю

```
IF IFNULL(AVG([Цена (Ноябрь)]), 0) != 0 
THEN (AVG([Цена (Декабрь)]) - AVG([Цена (Ноябрь)])) / AVG([Цена (Ноябрь)]) 
END
```

**Средний чек Декабрь** – взвешенная средняя цена (выручка / продажи)

```
IF IFNULL(SUM([Продажи (Декабрь)]), 0) != 0 
THEN SUM([Выручка (Декабрь)]) / SUM([Продажи (Декабрь)]) 
END
```

**Взвешенный выкуп Декабрь** – взвешенный % выкупа за декабрь

```
IF IFNULL(SUM([Продажи (Декабрь)]), 0) != 0 
THEN SUM([Продажи (Декабрь)] * [Средний % выкупа (Декабрь)]) / SUM([Продажи (Декабрь)]) 
END
```

**Взвешенный выкуп Ноябрь** – взвешенный % выкупа за ноябрь

```
IF IFNULL(SUM([Продажи (Ноябрь)]), 0) != 0 
THEN SUM([Продажи (Ноябрь)] * [Средний % выкупа (Ноябрь)]) / SUM([Продажи (Ноябрь)]) 
END
```

**Прирост взвешенного выкупа** – изменение взвешенного % выкупа в п.п.

```
(IFNULL([Взвешенный выкуп Декабрь], 0) - IFNULL([Взвешенный выкуп Ноябрь], 0)) * 100
```

**Прирост общего выкупа** – изменение общего % выкупа в п.п.

```
(AVG([Средний % выкупа (Декабрь)]) - AVG([Средний % выкупа (Ноябрь)])) * 100
```

**План выкупа выполнен** – флаг выполнения плана ≥95%

```
IF ROUND(IFNULL([Взвешенный выкуп Декабрь], 0), 2) >= 0.95 THEN 1 ELSE 0 END
```

**Взв % выкупа для точечного графика** – взвешенный % выкупа за декабрь с явной группировкой по уровню анализа (для корректной агрегации на точечном графике).

```
IF [analysis_type] = 'SKU' THEN 
  IF SUM([Продажи (Декабрь)]) != 0 
  THEN SUM([Продажи (Декабрь)] * [Средний % выкупа (Декабрь)]) / SUM([Продажи (Декабрь)]) 
  END
ELSEIF [analysis_type] = 'Бренд' THEN 
  IF SUM([Продажи (Декабрь)]) != 0 
  THEN SUM([Продажи (Декабрь)] * [Средний % выкупа (Декабрь)]) / SUM([Продажи (Декабрь)]) 
  END
ELSEIF [analysis_type] = 'Страна' THEN 
  IF SUM([Продажи (Декабрь)]) != 0 
  THEN SUM([Продажи (Декабрь)] * [Средний % выкупа (Декабрь)]) / SUM([Продажи (Декабрь)]) 
  END
ELSEIF [analysis_type] = 'Категория' THEN 
  IF SUM([Продажи (Декабрь)]) != 0 
  THEN SUM([Продажи (Декабрь)] * [Средний % выкупа (Декабрь)]) / SUM([Продажи (Декабрь)]) 
  END
ELSEIF [analysis_type] = 'Предмет' THEN 
  IF SUM([Продажи (Декабрь)]) != 0 
  THEN SUM([Продажи (Декабрь)] * [Средний % выкупа (Декабрь)]) / SUM([Продажи (Декабрь)]) 
  END
ELSE 
  IF SUM([Продажи (Декабрь)]) != 0 
  THEN SUM([Продажи (Декабрь)] * [Средний % выкупа (Декабрь)]) / SUM([Продажи (Декабрь)]) 
  END
END
```

**Анализ по срезам** – динамическая группировка для таблицы.

```
IF [analysis_type] = 'Бренд' THEN [Бренд]
ELSEIF [analysis_type] = 'Страна' THEN [Страна]
ELSEIF [analysis_type] = 'Категория' THEN [Категория]
ELSEIF [analysis_type] = 'Предмет' THEN [Предмет]
ELSEIF [analysis_type] = 'SKU' THEN
      [Бренд] + " | " + [Предмет] + " | " + STR([SKU])
ELSE [Категория]
END
```

**Датасет Dynamic**:

**Промежуточные поля**:

**Выкуп товаров** – количество выкупленных единиц:
```
[Продажи] * [Средний % выкупа]
```
**Выручка/Продажи/Выкуп (текущий месяц)** – значение метрики для выбранного месяца через параметр p_Selected_Month
```
SUM(IF [Месяц] = [p_Selected_Month] THEN [Выручка] END)
```
```
SUM(IF [Месяц] = [p_Selected_Month] THEN [Продажи] END)
```
```
SUM(IF [Месяц] = [p_Selected_Month] THEN [Выкуп товаров] END)
```
**Выручка/Продажи/Выкуп (предыдущий месяц)** – значение метрики для месяца, предшествующего выбранному
```
SUM(IF [Месяц] = DATEADD([p_Selected_Month], 'month', -1) THEN [Выручка] END)
```
```
SUM(IF [Месяц] = DATEADD([p_Selected_Month], 'month', -1) THEN [Продажи] END)
```
```
SUM(IF [Месяц] = DATEADD([p_Selected_Month], 'month', -1) THEN [Выкуп товаров] END)
```

**Логические флаги**:

**Данные за месяц** – TRUE, если выбран конкретный месяц (по умолчанию показываются данные за все месяцы)
```
[p_Selected_Month] != DATE('2016-01-01')
```
**Нет предыдущего месяца** – TRUE, если выбран июль 2023 (первый месяц в данных)
```
[p_Selected_Month] = DATE('2023-07-01')
```

**Метрики прироста (MoM)**:

**Прирост выручки** – процент изменения выручки относительно предыдущего месяца
```
IF [Выручка (предыдущий месяц)] != 0 
THEN ([Выручка (текущий месяц)] - [Выручка (предыдущий месяц)]) / [Выручка (предыдущий месяц)]
END
```

**Прирост продаж** – процент изменения количества продаж
```
IF [Продажи (предыдущий месяц)] != 0 
THEN ([Продажи (текущий месяц)] - [Продажи (предыдущий месяц)]) / [Продажи (предыдущий месяц)]
END
```

**Прирост общего выкупа** – изменение среднего арифметического % выкупа в п.п.
```
(AVG(IF [Месяц] = [p_Selected_Month] THEN [Средний % выкупа] END) 
- AVG(IF [Месяц] = DATEADD([p_Selected_Month], 'month', -1) THEN [Средний % выкупа] END))
* 100
```

**Прирост взвешенного выкупа** – изменение взвешенного % выкупа в п.п.
```
((IF [Продажи (текущий месяц)] != 0 THEN [Выкуп (текущий месяц)] / [Продажи (текущий месяц)] END)
  -
  (IF [Продажи (предыдущий месяц)] != 0 THEN [Выкуп (предыдущий месяц)] / [Продажи (предыдущий месяц)] END)) * 100
```

**Метрики для визуализации**:

**Выручка чарт / Продажи чарт** – переключаются между значением за месяц и суммой за 6 месяцев:

```
IF [Данные за месяц] 
THEN [Выручка (текущий месяц)] 
ELSE SUM([Выручка])
END
```
```
IF [Данные за месяц] 
THEN [Продажи (текущий месяц)] 
ELSE SUM([Продажи])
END
```

**Общий выкуп** – средний арифметический % выкупа (переключаются между значением за месяц и суммой за 6 месяцев)
```
IF [Данные за месяц]
THEN AVG(IF [Месяц] = [p_Selected_Month] THEN [Средний % выкупа] END)
ELSE AVG([Средний % выкупа])
END
```
**Взвешенный выкуп** – взвешенный % выкупа (переключаются между значением за месяц и суммой за 6 месяцев)
```
IF [Данные за месяц]
THEN IF [Продажи (текущий месяц)] != 0 THEN [Выкуп (текущий месяц)] / [Продажи (текущий месяц)] END
ELSE IF SUM([Продажи]) != 0 THEN SUM([Выкуп товаров]) / SUM([Продажи]) END
END
```

**Взвешенный % выкупа** – взвешенный % выкупа за все месяцы (для спарклайна динамики)
```
IF SUM([Продажи]) != 0 
THEN SUM([Продажи] * [Средний % выкупа]) / SUM([Продажи]) 
END
```
